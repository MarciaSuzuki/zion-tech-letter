# Zion Church Interactive Letter

This folder contains the interactive letter prepared for Zion Church.

It is ready to be published as a static site with GitHub Pages.

## Files

- `index.html`: main interactive letter
- `assets/`: fonts and images used by the page
- `carta-zion-interativa.pdf`: exported PDF version
- `preview.png`: preview image of the page

## GitHub Pages

If this repository is pushed to GitHub and Pages is enabled from the `main` branch using the root folder, the site will be available at:

`https://YOUR-USERNAME.github.io/REPOSITORY-NAME/`
