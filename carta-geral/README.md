# Carta Geral aos Irmãos e Mantenedores

This folder contains the general interactive letter prepared for churches,
brothers, and ministry supporters.

It can be shared as a local package or published later as a static site.

## Files

- `index.html`: main interactive digital letter
- `assets/`: fonts and images used by the page
- `carta-geral-impressao.html`: print-friendly HTML version
- `preview.png`: preview image of the page
- `carta-geral-interativa.pdf`: exported PDF from the digital version
- `carta-geral-impressao.pdf`: exported PDF from the print version
